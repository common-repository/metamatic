Simple friend list and friend feed functionality.

== Description ==

Adds Friends widget and Friend List page to your site.
Adds Friends Feed page that shows content loaded from friend's sites via RSS/ATOM.
Provides friends information in XML FOAF format.

* [Metamatic plugin page](http://metamatic.net/soft/metamatic-wp-en)

== Installation ==

1. Extract contents of the distribution archive to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
